# NeuraScent

Discover your perfect scent ritual with NeuraScent. Take our interactive quiz to elevate your everyday experience.

## Getting Started

- Visit the [live site](https://neurascent.vercel.app) (update with your Vercel URL after deployment).
- Explore the quiz at `/quiz`.

### Development Setup

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

To run the development server:

```bash
npm run dev
